T4 Templates (.tt) — это генераторы исходного кода в Visual Studio и .NET!

Что такое T4 (Text Template Transformation Toolkit):
Это технология, которая позволяет автоматически генерировать C# код на основе шаблонов. Вы пишете шаблон с логикой, а Visual Studio генерирует готовый .cs файл.

Простой пример T4 шаблона:
ChannelCellTemplate.tt — ШАБЛОН:
t4
<#@ template debug="true" hostspecific="true" language="C#" #>
<#@ output extension=".generated.cs" #>
<#@ parameter name="CellName" type="System.String" #>
<#@ parameter name="Capacity" type="System.Int32" #>
<#
    // Это C# код, который выполняется во время генерации
    string className = $"{CellName}ChannelCell";
    string busClassName = $"{CellName}ChannelBus";
#>
// Автоматически сгенерированный код из шаблона ChannelCellTemplate.tt
// Время генерации: <#= DateTime.Now #>

using System;
using System.Threading.Tasks;

namespace FractalCell.Generated
{
    public class <#= className #> : FractalCellTemplate<ChannelInternalBus, ChannelExternalBus>
    {
        private readonly int _capacity = <#= Capacity #>;
        
        public <#= className #>(
            CellConfiguration configuration,
            ChannelInternalBus internalBus,
            ChannelExternalBus externalBus)
            : base(configuration, internalBus, externalBus)
        {
        }

        protected override IEnumerable<BackgroundService> CreateBackgroundServices()
        {
            for (int i = 0; i < Configuration.BackgroundServiceCount; i++)
            {
                yield return new ChannelBackgroundService(
                    $"<#= CellName #>-worker-{i}",
                    InternalBus,
                    Logger,
                    Configuration);
            }
        }

        protected override void ConfigureHandlers()
        {
            InternalBus.Subscribe<FractalEvent>(async fractalEvent =>
            {
                Logger.LogInformation("<#= CellName #> received: {EventType}", 
                    fractalEvent.EventType);
                
                // Специфичная логика для <#= CellName #>
                await Process<#= CellName #>EventAsync(fractalEvent);
            });
        }

        private async Task Process<#= CellName #>EventAsync(FractalEvent @event)
        {
            // Кастомная обработка для этой ячейки
            Logger.LogInformation("Processing in <#= CellName #> cell with capacity <#= Capacity #>");
            
            switch (@event.EventType)
            {
                case "<#= CellName #>DataProcess":
                    // Специфичная обработка
                    break;
                default:
                    await ExternalBus.SendToCellAsync(@event.TargetCellId, @event);
                    break;
            }
        }
    }
}

Как использовать .tt файлы:
В Visual Studio: Правый клик на .tt файл → "Run Custom Tool"

В командной строке: TextTransform.exe ChannelCellTemplate.tt

В CI/CD: Автоматическая генерация перед сборкой

Преимущества T4 для вашей архитектуры:
Массовое создание ячеек: Один шаблон → сотни сгенерированных классов

Конфигурация из JSON: Меняете конфиг → перегенерируете код

Типобезопасность: Генерируется строго типизированный C# код

Шаблонизация поведения: Разные .tt для разных паттернов обработки

Это и есть та самая "фабрика фрактальных ячеек" — пишете конфиг, запускаете шаблон, получаете готовые классы! 🚀

При запуске шаблона с параметрами CellName = "Stratosphere" и Capacity = 5000 сгенерируется:
Где задаются эти параметры ?
Отличный вопрос! Параметры в T4 шаблонах можно задавать несколькими способами, в зависимости от того, как вы запускаете генерацию.

Способ 1: Передача параметров в коде (Runtime T4)
Program.cs или скрипт генерации:
csharp
using Microsoft.VisualStudio.TextTemplating;
using System.CodeDom.Compiler;

// Создаем экземпляр шаблона
var template = new ChannelCellTemplate();

// Создаем сессию и передаем параметры
var session = new Dictionary<string, object>();
session["CellName"] = "Stratosphere";
session["Capacity"] = 5000;
session["BackgroundServiceCount"] = 3;
session["EnableLogging"] = true;

// Инициализируем сессию шаблона
template.Session = session;

// Инициализируем шаблон
template.Initialize();

// Генерируем код
string generatedCode = template.TransformText();

// Сохраняем в файл
File.WriteAllText("Generated/StratosphereChannelCell.generated.cs", generatedCode);

Способ 4: Параметры из командной строки
Кастомный скрипт генерации generate-cells.ps1:
powershell
param(
    [string]$CellName = "DefaultCell",
    [int]$Capacity = 1000,
    [int]$Workers = 2
)

# Заменяем параметры в шаблоне
$templateContent = Get-Content "ChannelCellTemplate.tt" -Raw
$templateContent = $templateContent -replace '\$CellName\$', $CellName
$templateContent = $templateContent -replace '\$Capacity\$', $Capacity
$templateContent = $templateContent -replace '\$Workers\$', $Workers

# Сохраняем временный шаблон
$tempTemplate = "Temp_ChannelCellTemplate.tt"
$templateContent | Set-Content $tempTemplate

# Генерируем код
& "C:\Program Files\Microsoft Visual Studio\2022\Enterprise\Common7\IDE\TextTransform.exe" $tempTemplate

# Переименовываем результат
Move-Item "Temp_ChannelCellTemplate.generated.cs" "$CellNameChannelCell.generated.cs"
Remove-Item $tempTemplate

Самый удобный способ для вашего проекта:
Создайте cell-factory.ps1:
powershell
# Конфигурация всех ячеек
$cells = @(
    @{Name="Stratosphere"; Capacity=5000; Workers=3; Region="north"},
    @{Name="Troposphere"; Capacity=3000; Workers=2; Region="south"},
    @{Name="Mesosphere"; Capacity=7000; Workers=5; Region="equator"},
    @{Name="Ionosphere"; Capacity=8000; Workers=4; Region="space"}
)

foreach ($cell in $cells) {
    Write-Host "Generating $($cell.Name) cell..." -ForegroundColor Green
    
    # Создаем временный конфиг для текущей ячейки
    $config = @{
        CellName = $cell.Name
        Capacity = $cell.Capacity
        BackgroundServiceCount = $cell.Workers
        Region = $cell.Region
    }
    
    $config | ConvertTo-Json | Set-Content "generation-config.json"
    
    # Запускаем генерацию
    & TextTransform.exe ChannelCellTemplate.tt
    & TextTransform.exe TplCellTemplate.tt
    
    # Переименовываем результаты
    Move-Item "ChannelCellTemplate.generated.cs" "Generated/$($cell.Name)ChannelCell.cs" -Force
    Move-Item "TplCellTemplate.generated.cs" "Generated/$($cell.Name)TplCell.cs" -Force
}

Write-Host "All cells generated!" -ForegroundColor Cyan
